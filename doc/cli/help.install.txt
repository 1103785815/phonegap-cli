Usage: $0 install [options] <platform>

DEPRECATED:

  This command has been deprecated and will soon be removed entirely.

  You should now use the `$0 run <platform>` command.

  See `$0 run help` to learn more about the command.

Description:

  Installs the application on a specific platform.

  Automatically detects platform SDK support and uses the correct environment.

  See each environment for available [options].

Examples:

  $ $0 install android
  $ $0 install --device android

Also see:

  $0 local install <platform>
  $0 remote install <platform>
