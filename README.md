# PhoneGap Client [![Build Status][travis-ci-img]][travis-ci-url]

> PhoneGap and PhoneGap Build command-line interface and node.js library.

## Command-line

### Install

    $ npm install -g phonegap

### Usage

    $ phonegap help

## Node.js

### Usage

    var phonegap = require('phonegap');
